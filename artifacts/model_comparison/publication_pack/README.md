# publication_pack 输出说明

本目录是一键投稿证据包输出。

- ablation_main/: 主设定消融结果
- benchmark_models/: 同条件模型对比结果
- robustness_h3/: 预测周期 h=3 稳健性消融
- robustness_recent/: 近期样本稳健性消融
- significance_table.csv: 配对显著性检验结果
- publication_readiness_report.md: 自动生成投稿就绪评估

建议先阅读：

1. publication_readiness_report.md
2. ablation_main/ablation_summary.csv
3. significance_table.csv
